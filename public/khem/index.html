why khem
a language that
gets out of the way.
◈
Tcl-inspired syntax
Familiar command-and-block structure. No closing tags, no angle brackets. Structure emerges from indentation and braces.
⬡
Proc system
Define reusable components with proc. Pass arguments, compose freely, build design systems without a framework.
→
Compiles to standard HTML
Output is clean, readable HTML/CSS/JS. No runtime. No virtual DOM. Just files you can open in any browser.
⊕
Vite plugin
Drop vite-plugin-khem into any Vite project. Process type="text/khem" script tags at build time.
◇
Zero dependencies
The compiler is a single module. No heavy toolchain. Add it to your project in seconds, remove it just as fast.
⌘
Browser IDE
An embedded editor lets you write and preview Khem instantly. No install needed to explore the language.
